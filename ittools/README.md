#### *主要功能*
1. Crypto（加密）工具类， 包含token生成、哈希文本、UUID生成、加密解密文本等9个不同的功能。
1. Converter（转化）工具类，包含Yaml 转化器，Json转化器，Base64转化器等12种不同的功能。
1. 3.Web（网站）工具类，包含Url格式编码解码，用户代理解析器，网址解析器等15种不同的功能。
1. Mages and Videos （图片与视频）工具类，包含SVG 占位符生成器，二维码生成器等3种功能。
1. Development工具类，包含Docker run 到 Docker compose 转换器，定时任务生成器，SQL美化与格式化等10种不同功能。
1. 另外还有Network（网络）工具类，Math（数学）工具类，Measurement（测量）工具类，文本工具类，数据工具类等等
